# art/ — 카드 일러스트 폴더

카드 일러스트 이미지를 이 폴더에 넣습니다. 게임/개발자 페이지는 `../art-map.js`(매니페스트)를
통해 각 카드와 이미지를 연결합니다.

## 파일명 규칙 (slug)

| 카드 ID | 파일명 |
|---|---|
| `Fork` | `art/Fork.webp` |
| `Cache` | `art/Cache.webp` |
| `boost()` | `art/boost_fn.webp` |
| `copy()` | `art/copy_fn.webp` |

- 인스턴스(대문자 시작)는 ID 그대로.
- 포인터(`이름()`)는 `()` → `_fn`. (`Copy` 와 `copy()` 같은 대소문자 충돌 방지)
- 권장 포맷: `.webp`(용량↓). `.png` / `.jpg` 도 가능하나 매니페스트 경로 확장자를 맞출 것.
- 뷰포트는 가로형(≈150×92)·정사각(필드 타일)·모바일 미니 등 여러 비율로 crop 됩니다.
  **인물/핵심은 가운데**에 두고, 잘려도 되는 여백을 가장자리에 두면 안전합니다.

## 작업 방법

1. 프로젝트 루트에서 로컬 서버를 띄우고 `dev.html` 을 엽니다.
   ```
   python -m http.server 8000
   # → http://localhost:8000/dev.html
   ```
2. 카드를 고르고 이미지를 드래그&드롭 → 초점/맞춤을 조절 → **저장**.
   - Chrome/Edge: "폴더 연결" 후 이미지 파일과 `art-map.js` 가 자동 저장됩니다.
   - 그 외 브라우저: 이미지와 `art-map.js` 를 "내려받기" 해서 이 폴더/루트에 직접 넣습니다.
